HTML

Różnica między [b] & [i] a [strong] & [em]
- [b] & [i] - nie mają znaczenia semantycznego, ale chcemy, żeby tekst się wyróżniał
- [strong] & [em] - strong podkreśla istotność i poważność frazy, em podkreśla wypowiedź, np. ironia
- zaleca się używanie [strong] & [em]

Selektor class - grupuje elementy

Atrybut id - identyfikuje element (przydaje się do nawigacji wewnątrz strony)

img src alt -alt jest używane do np. czytników kodów, lub dla osób niewidomych, aby wiedzieli co jest na obrazku. Bardzo ważne w nowoczesnym budowaniu strony. Jeżeli zdjecie jest tylko dekoracją, to alt może być pusty.

CSS

Hierarchia specyficznosci:
I !important
II style="" - styl liniowy
III #name - ID
IV .name - klasa
V element

Żeby nie bawić się w specyficznośc, zaleca się dodawanie klasy do wszystkich elementów i unikać zagnieżdżeń

Jak działa przeglądarka? Kody odpowiedzi HTTP

100 - poczekaj
200 - wszystko jest OK
30x - 'szukasz w złym miejscu', przekierowania, przekierowujemy Cię w odpowiednie miejsce
404 - brak zasobu, nie ma tego czego szukasz
50x - wewnętrzny błąd serwera. Jako frontendowcy nie mamy wpływu

JavaScript

Będziemy uczyć się ECMA script 2015+ (ES6)

Składowe w JS:

  • Wartości - są to rzeczy, które już istnieją
  • Operacje porównywania
  • Zmienne
  • Literały
  • Funkcje
  • Obiekty

Zmienne w JS: const i let

const- rzeczy mające stałą strukturę, nie można ich podmienić

let- rzeczy, które można przypisać wielokrotnie, ale definiujemy tylko raz

${nazwaZmiennej} - zwraca zadeklarowaną zmienną

prawie wszystko w JS jest obiektem - tak się zachowują typy

Typy proste w JS (primitives):

  • String - tekst, otaczamy apostrofem '
  • Number - liczba
  • Bigint - liczba większa niż 253
  • Boolean - true lub false
  • null - wartość celowo niezidentyfikowana
  • undefined - brak definicji (błąd)
  • *symbol - tworzy "unikalne opakowanie". Drugi taki nie wystąpi

Polecenia w Windows

  • dir - wypisuje listę plików i folderów
  • cd 'nazwa' - otwiera dany katalog
  • mkdir 'nazwa' - tworzy folder
  • 'code .' - uruchamia Visual Studio Code będąc w danym folderze
